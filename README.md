# Glass-Flip-Clock-Countdown
A modern glass-style flip countdown clock built using HTML, CSS, and JavaScript.
This project features a smooth flip animation timer, glowing glassmorphism UI, and an animated gradient background for a visually appealing countdown experience.

The clock displays hours, minutes, and seconds, updating every second with a realistic flip animation effect.

Features 

1. Flip clock countdown animation
2. Glassmorphism UI design
3. Neon glow visual effects
4. Smooth animated gradient background
5. Real-time countdown updates using JavaScript
6. Responsive and lightweight design

Tech stack Used

1. HTML – Structure of the clock layout
2. CSS – Glassmorphism styling, animations, and effects
3. JavaScript – Countdown logic and flip animation updates

How to Run

1. Download the repository (A zip file will be downloaded).
2. Extract the ZIP file on your computer.
3. Open the extracted folder.
4. Double-click index.html or open it in any web browser.
5. The Glass Flip Clock Countdown will start running automatically.

Note: Make sure all files (index.html, style.css, script.js) are in the same folder.
